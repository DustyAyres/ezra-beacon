Sidebar Debug Test

Current Issue:

When clicking hamburger menu on mobile, screen turns grey but sidebar is not visible.

Debug Steps:

  1. Open browser DevTools (F12)
  2. Switch to mobile view (Ctrl+Shift+M)
  3. Set width to 375px
  4. Click hamburger menu
  5. Check Elements tab for:
    • Is body tag getting sidebar-open class?
    • Is .sidebar element getting sidebar-open class?
    • What is the computed transform value of .sidebar?
    • What is the computed z-index of .sidebar?

Expected CSS Values:

.sidebar {
    z-index: 9999;
    transform: translateX(0); /* when open */
    background-color: white;
    position: fixed;
    left: 0;
    top: 0;
    width: 80%;
}

.sidebar-overlay {
    z-index: 998;
    display: block; /* when sidebar is open */
}

Test in Mobile Frame:

Common Issues to Check:

  • Transform not applying: Check if transform: translateX(0) is being applied when sidebar-open class is added
  • Z-index conflict: Another element might have higher z-index
  • Display none: Sidebar might be hidden with display: none somewhere
  • Width issue: Sidebar might have 0 width
  • Overflow hidden: Parent container might be hiding the sidebar

Console Commands to Debug:

// Check if sidebar exists
document.querySelector('.sidebar')

// Check sidebar classes
document.querySelector('.sidebar').className

// Check computed styles
getComputedStyle(document.querySelector('.sidebar')).transform
getComputedStyle(document.querySelector('.sidebar')).zIndex
getComputedStyle(document.querySelector('.sidebar')).display
getComputedStyle(document.querySelector('.sidebar')).width

// Check if body has sidebar-open class
document.body.classList.contains('sidebar-open')

// Manually toggle sidebar
document.querySelector('.sidebar').classList.add('sidebar-open')
